Password generator

You will need to create a program that randomly generates a passwords.
The passwords need to be in the following form of an adjective, a noun and a number between 1 and 100
e.g. roundbird28

Tasks

  1. import the random module
  2. make a list of at least 10 nouns make sure that the words have " "and that each word is separted by a comma
  3. make a list of at least 10 adjectives
  4. generatre a random number between 1 and 100
  5. randomly choose an adjective
  6. randomly choose a noun
  7. join the adjective, noun and number to make a password Use the + to join strings
