Import

It is useful to use other peoples code so we can focus on solving our problems without having to invent everything ourselves

The import command imports a module with someone elses code

The example to the left imports the random module so that we can get random numbers. To use the randint command from the random module we need the following syntax:
random.randint(1,6)
This generates a random number between 1 and 6
