Lists and choice

Lists

Lists are a way of holding data. The data in a list is separated by commas ,. The following is an example of a list of strings (characters)

['certain','outlook not so good', 'impossible']

choice

The choice function in the random module randomly chooses an element (thing) from a list it's syntax is as follows:
random.choice(listName)
