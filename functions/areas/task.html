Maths Formulas

You can do maths with Python, the basic operations are:

  • + addition
  • - subtraction
  • * mutiplication
  • / division
  • ** power (e.g. 42 would be 4**2)
  • % modulo, gives the remainder of a division

The math module includes other maths functions. We will use it to get the value of pi so that we can find the area of a circle.
To get pi use the command math.pi

Task

You need to create functions to that return the area of a square, a rectangle, a triangle and a circle.
