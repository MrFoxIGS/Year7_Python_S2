Project: Text Based Adventure Game

Specifications

You need to create a text based adventure game
You game should:

  • Contain at least 5 scenes
  • Each scene needs at leat 2 choices
  • You need to be able to finish the game
  • Your code should be easy to follow with the use of comments, docstrings and white space
  • You should test your game and improve it with the end user in mind

An example of how you can use functions to make your game is given

Before you begin you should complete your design on paper so that you know what scenes will be in you game and how they' are connected to each other
